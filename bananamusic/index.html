About BananaMusic

EzMusic was a music bot using jagrosh's JMusicBot, a music bot that is easy to host yourself!

However, Wicycool shut down EzMusic on September 5th, 2021, due to him wanting to use his VPS for other purposes. So, I remade it, using my own VPS! It's still using all the same code, so all the commands below and such are still valid.

BananaMusic is only in Kevintendo's Discord server at the moment, and is not available in any other servers.

Differences between Fredboat

Because this bot replaces Fredboat, some changes were made to make the commands more similar. These can be found in the Command aliases section below. However, there are some major differences you needed to know:

When you leave the Music VC

When you're by yourself and you leave the Music VC, PLEASE and I mean PLEASE ALWAYS STOP THE BOT BEFORE YOU LEAVE. If you leave it playing, it will keep playing until the music stops, even if no one's in the VC. To save my VPS' bandwith, please just use ;;stop before you leave the VC.

Difference between ;;play and ;;p

;;p is used to SEARCH for things, and ;;play is used to DIRECTLY LOAD FROM LINKS. This means that you should do something like this:

;;play https://www.youtube.com/watch?v=dQw4w9WgXcQ
- loading a song directly from a link
;;p Rick roll
- loading a song by searching

Difference between ;;skip and ;;s

;;skip is used to do a VOTE skip, while ;;s FORCE skips it, without needing any votes. You will need to so something like this:

;;skip
- Add a vote to skip a song
;;s
- Force the current ssong to be skipped

Commands

Music

;;aboutshows info about the bot
;;ping checks the bot's latency
;;settings shows the bots settings
;;lyrics [song name] shows the lyrics to the currently-playing song
;;nowplayingshows the song that is currently playing
;;play <title|URL|subcommand> plays the provided song
;;queue [pagenum] shows the current queue
;;remove <position|ALL>removes a song from the queue
;;search <query>searches Youtube for a provided query
;;scsearch <query> searches Soundcloud for a provided query
;;shuffle shuffles songs you have added
;;skip votes to skip the current song

DJ

;;forceremove <user>removes all entries by a user from the queue
;;forceskip skips the current song
;;movetrack <from> <to> move a track in the current queue to a different position
;;pause pauses the current song
;;playnext <title|URL>plays a single song next
;;repeat [on|off] re-adds music to the queue when finished
;;skipto <position>skips to the specified song
;;stop stops the current song and clears the queue
;;volume [0-150]sets or shows volume

More commands can be found by using the ;;help command.
